四、精神科護理學 — 必考重點清單

以下為精神科護理學歷年反覆出現的核心考點，建議逐項確認掌握程度。

治療性溝通與護病關係

  • 治療性溝通技巧（同理心、反映、澄清、面質、集中話題）
  • 情感轉移與情感反轉移（transference / countertransference）的定義與處理
  • Peplau 護病關係四階段（認識前期 → 認識期 → 運用期 → 解決期）
  • 門寧格態度治療（主動友善 / 被動友善 / 就事論事 / 寬容 / 監控）

精神科藥物治療

  • 抗精神病藥物（第一代 vs 第二代、錐體外徑症候群、遲發性運動不能、惡性症候群）
  • Clozapine 特殊副作用（白血球下降 → 定期血液監測、代謝症候群）
  • 情緒穩定劑（鋰鹽血中濃度 0.6-1.2 mEq/L、毒性症狀、Valproic acid、Carbamazepine/SJS）
  • 抗憂鬱劑分類（SSRI、三環、SNRI）及副作用
  • BZD 類藥物（抗焦慮、助眠、耐受性成癮性、酒精戒斷使用）
  • 電氣痙攣療法（ECT）適應症、護理（午夜後禁食、簽同意書）、併發症

思覺失調症與情緒障礙

  • 思覺失調症正性 vs 負性症狀、多巴胺假說、病程（前驅期 → 發病期 → 殘留期）
  • 憂鬱症護理（自殺防範優先、服藥衛教、漸進活動安排）
  • 躁症護理（環境簡化、高熱量飲食、設限、消耗體力活動）

焦慮症與物質使用障礙

  • 焦慮四級別（輕度 → 中度 → 重度 → 恐慌）特徵與護理
  • 恐慌症發作處理（安全環境、呼吸放鬆、紙袋）
  • 強迫症（SSRI 藥物治療、ERP 暴露與反應阻止法）
  • PTSD（flashback 回憶重現、逃避、過度警覺、持續 > 1 個月）
  • 酒精戒斷（震顫性譫妄、維生素 B1 缺乏 / Wernicke-Korsakoff、BZD 治療）
  • 海洛因中毒（瞳孔縮小、呼吸抑制）、美沙冬替代療法
  • 安非他命戒斷（耗竭虛脫、憂鬱沮喪）

認知障礙與人格障礙

  • 失智症 vs 譫妄鑑別（急性 vs 慢性、意識清 vs 波動、可逆 vs 不可逆）
  • 認知障礙症護理（簡短溝通、定向感訓練、安全環境、轉移注意力）
  • 人格障礙症（邊緣型：分裂 / 全好全壞、反社會型：操控、畏避型：害怕批評）

安全管理與社區復健

  • 心理防衛機轉（潛抑、投射、合理化、昇華、抵銷、反向、退化、隔離、內射）
  • 精神衛生法（強制鑑定、強制住院、病人權益保障）
  • 約束處置（醫囑、每 15 分鐘探視、每 2 小時評估解除）
  • 自殺防範（評估自殺意念與計畫、移除危險物品、密切觀察、病情好轉期最危險）
  • 三級預防（初級：心理衛生教育 / 次級：早期發現治療 / 三級：復健回歸社區）
  • 團體治療（療效因素：利他、普遍性、人際學習、凝聚力）
  • Caplan 危機理論四期（衝擊 → 紊亂 → 求助 → 崩潰）
  • 精神社區復健資源（日間留院、社區復健中心、康復之家、精神護理之家）

六、社區衛生護理學 — 必考重點清單

以下為社區衛生護理學歷年反覆出現的核心考點，建議逐項確認掌握程度。

預防醫學與流行病學

  • 三段五級預防（初段：健康促進 + 特殊保護 / 次段：早期發現 + 限制殘障 / 末段：復健）
  • 流行病學研究方法（描述性、橫斷性、世代研究、病例對照、臨床試驗）
  • 篩檢工具效度（敏感度 = 真陽性率、特異度 = 真陰性率、陽性預測值、陰性預測值）
  • 盛行率 vs 發生率的差異與用途
  • 生命統計指標（粗死亡率、嬰兒死亡率、新生兒死亡率、依賴指數、老化指數）

篩檢與預防保健

  • 四癌篩檢給付條件（子宮頸癌 30+ / 乳癌 45-69 每 2 年 / 大腸癌 50-74 每 2 年 / 口腔癌）
  • 成人預防保健服務（40-64 歲每 3 年 / 65 歲以上每年）
  • 代謝症候群五項指標（腰圍 / 血壓 / 空腹血糖 / 三酸甘油酯 / HDL）

傳染病防治

  • 傳染病分類（第一類 ~ 第五類）與通報時限
  • 登革熱（埃及斑蚊 / 白線斑蚊、布氏指數、第二類傳染病、潛伏期 3-14 天）
  • 結核病（空氣傳染、都治計畫 DOTS、潛伏感染無傳染力）

健康促進理論

  • 健康信念模式六要素（疾病易感 / 嚴重性認知、行動利益 / 障礙認知、行動線索、自我效能）
  • 渥太華健康促進憲章五大行動綱領
  • 社會生態模式（個人 → 人際 → 組織 → 社區 → 政策）

社區評估與健康促進

  • Anderson 社區夥伴評估模式（八大系統）
  • 社區健康評估方法（走街式調查、重要人物訪談、民意調查、參與式觀察、德菲法）

長期照顧與居家護理

  • 長照 2.0 ABC 三級服務（A: 社區整合服務中心、B: 複合型服務中心、C: 巷弄長照站）
  • 長照 2.0 服務對象與申請條件
  • 小規模多機能服務（日照 + 居家 + 臨時住宿）
  • 喘息服務（輕中度 14 天 / 重度 21 天 / 年）
  • 居家護理（全民健保給付、鼻胃管 / 氣切 / 尿管照護、每月 2 次訪視）
  • 出院準備服務（入院 24 小時內、跨專業團隊、出院當日非結案日）

家庭護理與評估

  • APGAR 家庭功能評估（適應度 / 合夥度 / 成長度 / 情感度 / 融洽度）
  • Duvall 家庭發展階段理論（8 個階段）
  • 家庭評估工具（家系圖、生態圖、家庭圈）

環境職業衛生與其他

  • 環境衛生（自來水曝氣法、氯氣消毒、酸雨 pH < 5.6、PM2.5）
  • 食品安全（金黃色葡萄球菌潛伏期最短、肉毒桿菌真空食品）
  • 職業衛生（噪音 85dB 防護、TWA 八小時容許濃度）
  • 學校衛生（40 班以上護理人員 2 名、每學期身高體重視力）
  • SOF 衰弱評估（三項：體重減輕 / 精力下降 / 下肢功能，≥ 2 項為衰弱）
  • 全民健保（強制性社會保險、部分負擔防 moral hazard）

七、歷年趨勢變化

  • 題數改制：111 年第一次至 112 年第二次為舊制 80 題（每題 1.25 分），112 年第三次起改為新制 50 題（每題 2 分），總分仍為 100 分。題數減少但每題分值加重，答錯代價更高。
  • 長照 2.0 題數顯著增加：隨著長照政策持續推動，長照 2.0 相關考題逐年增加，近三年平均每次考試出現 5-7 題，涵蓋 ABC 服務體系、喘息服務、小規模多機能等新型態服務。
  • 老年精神護理比重上升：因應高齡化社會，認知障礙症（失智症）相關考題微幅增加，特別著重失智症 vs 譫妄的鑑別診斷，以及社區失智照護模式。
  • 精神衛生法修法反映：精神衛生法的修訂內容已反映在近年考題中，包括強制住院流程的變更、社區追蹤關懷制度、病人自主權等議題。
  • 整合型情境題增加：跨領域的臨床情境題越來越多，例如結合精神科藥物與護理措施、社區資源轉介與長照服務銜接等，要求考生具備整合應用能力。
  • 社區衛生時事融入：COVID-19 疫情期間的社區防疫、居家檢疫措施，以及後疫情時代的心理健康議題，已出現在近年考題中。
  • 老人健康議題受重視：跌倒預防、衰弱症篩檢（SOF 量表）、肌少症、多重用藥等老人健康議題出題頻率逐漸增加。

八、考前衝刺策略

最高優先

第一階段：必拿分（約佔 45%）

  • 思覺失調症正/負性症狀與護理
  • 精神科藥物（抗精神病藥、鋰鹽、抗憂鬱劑）
  • 治療性溝通技巧與護病關係
  • 情緒障礙（憂鬱/躁症）護理重點
  • 長照 2.0 ABC 三級服務體系
  • 流行病學研究方法與篩檢效度
  • 三段五級預防完整架構
  • 四癌篩檢給付條件
次高優先

第二階段：高頻考點（約佔 30%）

  • 焦慮症分類與護理（恐慌/強迫/PTSD）
  • 物質使用障礙（酒精/海洛因/安非他命）
  • 失智症 vs 譫妄鑑別
  • 社區健康評估方法
  • 傳染病分類與通報
  • 健康信念模式/渥太華憲章
  • 家庭評估工具（APGAR/家系圖）
  • 環境與職業衛生
  • 居家護理與出院準備
基礎鞏固

第三階段：中頻考點（約佔 15%）

  • 人格障礙症分類與護理
  • 兒童青少年精神疾病
  • 心理防衛機轉辨識
  • 學校衛生護理規範
  • 全民健保制度
  • 社區營造與社區資源
  • 婦幼衛生與營養

複習策略建議

  • 倒數 30 天：集中火力在「最高優先」的 8 大考點，每天至少讀完 1 個主題並做歷屆相關題目。精神科與社區各半交替進行。
  • 倒數 14 天：進入「次高優先」考點，同時每天回顧一個已讀主題的必考重點清單。
  • 倒數 7 天：做 3-4 回完整模擬考（新制限時 50 分鐘/50 題），錯題分類檢討，區分精神科與社區各自弱點。
  • 倒數 3 天：只看必考重點清單與錯題本，不再接觸新內容。特別確認藥物劑量、法規數字等易混淆項目。
  • 考前一天：輕鬆瀏覽高頻表格，早點休息。信心比知識更重要。

九、四週密集讀書計畫

以下為建議的四週密集複習計畫，每週聚焦不同主題領域，最後一週進行總複習與模擬考。

Week 1：精神科基礎（溝通技巧 + 藥物治療 + 思覺失調症 + 情緒障礙）
Day 1-2
治療性溝通與護病關係 — Peplau 四階段、溝通技巧（同理心/反映/澄清/面質）、情感轉移與反轉移、門寧格態度治療。搭配歷屆溝通情境題練習。
Day 3-4
精神科藥物治療 — 第一代 vs 第二代抗精神病藥、EPS/TD/NMS、Clozapine 監測、鋰鹽（0.6-1.2 mEq/L）、抗憂鬱劑（SSRI/TCA/SNRI）、BZD、ECT。製作藥物比較表。
Day 5-6
思覺失調症護理 — 正性/負性症狀、多巴胺假說、病程三期、護理計畫。情緒障礙症護理 — 憂鬱症（自殺防範優先）、躁症（環境簡化、設限）。
Day 7
