領域配比說明

本科目合併「基本護理學（護理原理、護理技術）」與「護理行政」兩大領域，在每次考試中大約維持 6:4 的比例分配。 80題制時期約 48 題基護 + 32 題行政；50題制時期約 30 題基護 + 20 題行政。 兩領域性質截然不同：基本護理學偏重技術操作與臨床情境判斷，護理行政偏重理論應用與管理計算。 備考策略應兼顧兩者，不可偏廢。

給藥

💊 給藥（Medication Administration） 佔比約 8.3% ｜ 每次必考 3-5 題

出題重點分析

主題出題頻率重要度常見考點
給藥途徑與醫囑縮寫 每次必考 ★★★★★ PO/SL/SC/IM/IV 途徑辨識；QD/BID/TID/QID/PRN/QOD 頻率；A.U./A.S./A.D. 眼耳用藥縮寫
給藥五對/三讀五對 每次必考 ★★★★★ 病人對、藥物對、劑量對、途徑對、時間對；三讀時機（取藥、倒藥、歸位）
藥物劑量計算 每次必考 ★★★★★ 劑量換算（mg/mL計算抽取量）、點滴速率計算（gtt/min, mL/hr）、熱量計算
注射技術 高頻 ★★★★ IM注射部位（腹臀肌/三角肌/股外側肌）、SC注射角度（45-90度）、Z-track技術、胰島素注射
特殊給藥方式 高頻 ★★★ 舌下含服（NTG）、肛門栓劑（直腸黏膜吸收）、吸入劑（MDI使用後漱口防念珠菌）、管灌給藥（不可磨碎膠囊/緩釋劑型）

給藥必背要點

  • NTG 舌下含服：胸痛時使用，每5分鐘可重複一次，最多3次
  • Gentamycin 40mg/mL，需100mg → 抽取2.5mL
  • 持續性藥效膠囊（如 tacrolimus）整顆吞服，不可磨碎/剝半/咬碎
  • Corticosteroids 乾粉吸入劑使用後需漱口→避免口腔念珠菌感染
  • 肛門栓劑主要經直腸黏膜吸收產生藥效
  • G5W 1000mL 提供 50g 葡萄糖 = 200 Kcal；G5W 1000mL + G5S 500mL = 75g = 300 Kcal
  • 點滴速率計算：mL/hr = 總量(mL) / 總時數(hr)；170mL/day ÷ 24hr ≈ 7mL/hr

體液/輸液/輸血

🩸 體液電解質 / 輸液 / 輸血 佔比約 6.9% ｜ 每次必考 2-4 題

出題重點分析

主題出題頻率重要度常見考點
靜脈輸液併發症 每次必考 ★★★★★ 血栓靜脈炎（移除原針、抬高患肢）、滲漏、空氣栓塞、循環負荷過重
輸血注意事項 每次必考 ★★★★★ 最初15-30分鐘慢速（20-40 gtt/min）；每單位2小時內輸完；加壓器壓力不超過300mmHg；加溫器38-40℃；血品領取後不放護理站冰箱
體液電解質平衡 高頻 ★★★★ 低血鉀/高血鉀症狀與處置、脫水評估、水腫評估、電解質補充
酸鹼平衡 高頻 ★★★ 代謝性酸中毒/鹼中毒、呼吸性酸中毒/鹼中毒、ABG判讀

輸液/輸血必背要點

  • 血栓靜脈炎處置：立即移除原注射針→抬高注射部位（非按摩、非濕冷敷）
  • 輸血前15-30分鐘滴速 20-40 gtt/min，無反應後按醫囑調整
  • 輸血加壓器壓力 ≤ 300 mmHg（非400）
  • 輸血加溫器溫度 38-40℃
  • 血品離開血庫超過30分鐘不可退回；血漿收到後儘速輸完

感染控制 / 無菌技術

🧬 感染控制與無菌技術 佔比約 5.6% ｜ 每次必考 2-3 題

出題重點分析

主題出題頻率重要度常見考點
內外科無菌技術區別 每次必考 ★★★★★ 內科無菌=清潔技術（洗手指尖朝下、低於肘部）；外科無菌=滅菌技術（應用於侵入性治療如導尿、術後傷口換藥）
隔離措施 每次必考 ★★★★★ 肺結核→空氣傳染防護；疥瘡→接觸傳染防護；骨髓移植→保護性隔離；化療病人→保護性隔離
手部衛生 高頻 ★★★★ 戴手套不能取代洗手；沾染血液/體液後須用肥皂水洗手（非酒精乾洗手）；指甲尖端 ≤ 0.5cm
消毒劑/終期消毒 高頻 ★★★ 75%酒精消毒效果優於95%；3%雙氧水可去除血漬痂皮；終期消毒：有分泌物血跡以漂白水清洗；紫外線燈照射

感染控制必背要點

  • 外科無菌技術應用於：導尿管放置、術後傷口換藥等侵入性治療
  • 肺結核 = 空氣傳染防護（N95口罩）；疥瘡 = 接觸傳染防護
  • 沾染血液/體液 → 必須用肥皂水洗手，不可僅用酒精乾洗手
  • 終期消毒：血跡/分泌物以漂白水清洗擦拭；布單先清洗再消毒
  • 傳染病大流行 → 隔離病房宜採功能性護理模式

生命徵象

🩺 生命徵象（Vital Signs） 佔比約 5.6% ｜ 每次必考 2-4 題

出題重點分析

主題出題頻率重要度常見考點
體溫測量與發燒型態 每次必考 ★★★★★ 間歇熱（敗血症）、弛張熱（波動＞2℃）、稽留熱（波動＜1℃，大葉性肺炎）、回歸熱（傷寒）；基礎體溫排卵後上升0.3-0.6℃（黃體素）
血壓測量 高頻 ★★★★ 膝膕動脈血壓測量（壓脈帶大腿中段前側）；膝膕動脈收縮壓 > 肱動脈；高壓脈vs低壓脈
脈搏與呼吸 高頻 ★★★ 出血性休克→低壓脈；動脈硬化→高壓脈；喟嘆式呼吸→情緒反應；測量呼吸不可先告知病人
影響體溫因素 高頻 ★★★ 新陳代謝增加→體溫升高；老人皮下脂肪薄→體溫較低；晝夜節律：傍晚4-8時最高；排卵後黃體素→基礎體溫上升

生命徵象必背要點

  • 間歇熱：體溫反覆升高又降至正常以下 → 敗血症
  • 弛張熱：24hr波動 ＞ 2℃ → 術後傷口感染（非傷寒）
  • 稽留熱：24hr波動 ＜ 1℃ → 大葉性肺炎
  • 回歸熱：高熱期與退熱期交替 → 肺結核（非傷寒）
  • 出血性休克 → 低壓脈（非高壓脈）
  • 排卵後基礎體溫平均上升 0.3-0.6℃
  • 雙臂骨折上石膏 → 改測膝膕動脈血壓，壓脈帶置於大腿中段前側

出題重點分析

主題出題頻率重要度常見考點
存留導尿 每次必考 ★★★★★ 女性導尿管潤滑5-7cm；水球注入5-10mL無菌蒸餾水；先鋪洞巾再消毒；導尿管大小：男性14-16Fr，女性12-14Fr

冷熱療法

🌡 冷熱療法（Cold/Heat Therapy） 佔比約 4.9% ｜ 每次必考 2-3 題

出題重點分析

主題出題頻率重要度常見考點
用熱生理效應 每次必考 ★★★★★ 降低血液黏稠度、增加微血管通透性、增加組織代謝率、促進白血球活動；扭傷48hr後熱敷→促進血液循環
冷熱共同效果 高頻 ★★★★ 冷與熱皆可減緩疼痛（共同效果）；冷：減緩炎症反應/充血腫脹；熱：減緩關節僵硬
溫水坐浴 高頻 ★★★★ 會陰傷口41-43℃溫水；主要目的：促進傷口血液循環；浸泡約20分鐘（過長→低血壓）；頭暈→身體前傾、頭低於心臟
耐受性因素 高頻 ★★★ 面積愈大→耐受力愈低；時間愈長→耐受力愈低；老人小孩耐受力較低（非較高）；神經感覺不良→易造成傷害
烤燈使用 中頻 ★★ 屬於輻射熱傳導（非對流）；25瓦距皮膚14-18吋；使用前清潔皮膚但不塗凡士林

冷熱療法必背要點

  • 用熱效應：降低血液黏稠度、增加微血管通透性、增加組織代謝（非減少）
  • 冷與熱的共同效果 = 減緩疼痛
  • 扭傷處理：48小時內冰敷 → 48小時後熱敷促進血液循環
  • 坐浴後頭暈/心跳加快/蒼白 = 姿勢性低血壓 → 身體前傾、頭低於心臟
  • 暴露面積大/時間長 → 耐受力降低；老人小孩耐受力較低

營養 / 管灌

🍽 營養評估與管灌餵食 佔比約 4.9% ｜ 每次必考 2-3 題

出題重點分析

主題出題頻率重要度常見考點
鼻胃管灌食 每次必考 ★★★★★ 矽膠材質7-10天更換；打入空氣確認位置後需抽出空氣；長期禁食後宜採連續性灌食；拔管時高坐臥、發「阿」聲吐氣；每日旋轉管路約45度
飲食種類 高頻 ★★★★ 清流質（大腸鏡前）：蜂蜜水、去油清湯、無渣果汁（非含渣、非排骨湯、非牛奶、非豆漿）；高蛋白飲食：50kg × 1.5-2g = 75-100g
營養評估指標 高頻 ★★★ 血清白蛋白反映過去21天蛋白質代謝；運鐵蛋白在肝臟製造（非腎臟）；BMI計算：55/(1.6)²=21.5→正常範圍
六大營養素 中頻 ★★★ 脂肪氧化需醣類參與，否則產生過多酮體→酮酸中毒；蛋白質由胺基酸組成，人體無法自行合成者=必需胺基酸

營養/管灌必背要點

  • 清流質飲食：蜂蜜水、去油清湯、無渣米湯（排除：牛奶、含渣果汁、排骨湯、豆漿）
  • 運鐵蛋白在「肝臟」製造（非腎臟）→慢性營養不良評估指標
  • 血清白蛋白→反映過去21天體內蛋白質代謝狀況
  • 鼻胃管矽膠材質 → 每7-10天更換一次
  • 安寧病人自拔鼻胃管→護理計畫：跟病人共同討論可行方法（非視為不遵從）
  • 睡眠中促進蛋白質合成與組織修復 → NREM第四期

活動與擺位

🏋 活動、擺位與身體力學 佔比約 4.2% ｜ 每次必考 2-3 題

出題重點分析

主題出題頻率重要度常見考點
翻身擺位原則 每次必考 ★★★★★ 保持頭頸脊椎一直線；肢體勿重疊；身體凹陷處勿懸空（需支托）；側臥加強肩部髖部皮膚護理；移動時不用手抓握肢體

  • Peter Drucker 目標管理(MBO) = 參與式管理（非權威式/放任式/民主式）

  • 傳統 vs 修正理論主要差異：參與式決策和團體價值觀
  • Gillies管理五大步驟中「規劃」包含：制定標準、擬定政策、作成決策、編列預算

人力 / 排班 / 護理模式

👥 護理人力、排班與照護模式 佔比約 8.7% ｜ 每次必考 3-5 題

出題重點分析

主題出題頻率重要度常見考點
護理照護模式 每次必考 ★★★★★ 功能護理（按技術分工、新進多）；成組護理（經驗少人力多時適用）；全責護理（一位護理師全權負責）；技術混合照護（不同能力人員搭配）
人力計算/缺勤率 每次必考 ★★★★★ 缺勤率計算：請假天數/(員工數×工作天數)×100%；護病比；病人分類系統(PCS)功能；人力彈性運用策略

人力彈性策略 高頻 ★★★ 數量彈性（增減人力）；職能彈性（技術混合照護）；時間彈性（工作分享制、彈性班別）

人力/排班/模式必背要點

  • 功能護理 = 按技術/任務分工、以工作為中心 → 新進護理師多時適用
  • 成組護理 = 經驗豐富護理師帶領小組 → 資淺多/臨床經驗少時適用
  • 傳染病大流行 → 隔離病房宜採功能性護理
  • 缺勤率 = 請假天數 / (員工數 × 工作天數) × 100%；如：60 / (100×240) × 100% = 0.25%
  • 病人分類系統(PCS)可作為：護理收費標準、評估績效、滿足照護需求（非病人預後指標）
  • 數量彈性=增減人力；職能彈性=技術混合；時間彈性=工作分享/彈性班別

  • 觸角效應/近因效應 = 考核前發生爭執而給予較低分數
  • 月暈效應 = 因某一優點概括評定所有面向
  • 集權式組織優點 = 減少工作重複及成本（非員工自由受尊重）
  • 分權化程度高 → 扁平式結構 → 員工自主權多
  • Jaques：W＞C＞P（工作量>能力>報酬）→ 員工有被剝削懲罰感

  • 證照租借他人使用 → 廢止「護理人員證書」

  • 損益平衡點 = 固定成本 / (1-變動成本/營收) → 400萬/(1-600/1000) = 1000萬

  • Benner：到職2.5年→勝任者(competent)（非精通者/進階學習者）
  • N1→N2進階訓練：教學原理與方法 + 危機處理
  • 5W1H訂工作目標缺乏「When」→ 缺乏完成時間性
  • 護理人力成本計算：月薪/月工時 × 活動時數 = 45000/180 × 1.5 = 375元

歷年趨勢變化

命題趨勢觀察

111年~112年前期（80題制）

題數多，涵蓋面廣，較多基礎知識記憶型考題。管理理論考得較為基礎，多為「是誰提出的」「屬於哪個理論」等辨識型題目。

112年第三次（50題制開始）

改制後每題2分，比重倍增。開始出現更多情境應用題，需要理解後判斷而非純記憶。給藥劑量計算題維持穩定出題。

113年~114年

臨床情境題比重明顯增加，如：安寧緩和病人自拔管路的護理處置、傳染病大流行時的照護模式選擇。病人安全與通報系統題目穩定出現，呼應實務重視程度。

114年第三次~115年第一次（最新趨勢）

整合性情境題增多，一題常跨越基護與行政兩領域。法規題更貼近實際場景（如哺乳期擠乳、醫療糾紛處理）。成本計算/人力計算等數學題穩定1-2題。

  • 肺結核→空氣傳染防護；疥瘡→接觸傳染防護；骨髓移植→保護性隔離
  • 沾染血液體液 → 用肥皂水洗手（非酒精乾洗手）
  • 主觀資料 = 病人主訴（如「感覺頭痛」）；客觀資料 = 觀察測量值
  • 護理問題排序依Maslow：呼吸困難 > 疼痛 > 睡眠 > 心像
  • 間歇熱=敗血症；弛張熱=術後感染；稽留熱=大葉性肺炎；回歸熱=肺結核
  • 輸血前15-30分鐘慢速20-40gtt/min；加壓器≤300mmHg；加溫器38-40℃
  • 血栓靜脈炎→立即移除原針頭、抬高患肢
  • 用熱效應：降低血液黏稠度、增加微血管通透性（非降低）
  • 冷熱共同效果 = 減緩疼痛
  • 坐浴溫度41-43℃；浸泡約20分鐘；頭暈=姿勢性低血壓→頭低於心臟
  • 清流質 = 蜂蜜水/去油清湯/無渣米湯（非牛奶/豆漿/含渣果汁）
  • 鼻胃管矽膠材質7-10天更換；每日旋轉約45度
  • 壓力性損傷預防：鼓勵主動運動（不可用氣墊圈/不可按摩/不可烤燈）
  • 穿脫衣服：脫健側先、穿患側先
  • 存留導尿：女性潤滑5-7cm、水球注入5-10mL無菌蒸餾水
  • 腹部按摩促排便：由「右至左」順結腸方向
  • Kubler-Ross：否認→憤怒→討價還價→憂鬱→接受（不一定按順序）
  • Berlo SMCR 溝通模式：發訊者→訊息→管道→收訊者

護理行政 Top 15 必背

  • Taylor 科學管理=工作標準化+時間動作研究；Fayol=管理程序(PODC)
  • X理論=人性本惡消極；Y理論=人性本善積極；Z理論=日本團隊共識
  • Peter Drucker 目標管理(MBO)=參與式管理
  • 功能護理=按任務分工；成組護理=小組合作；全責護理=一人全權負責
  • 傳染病大流行隔離病房 → 功能性護理模式
  • 情境領導：低能力低意願→告知式(S1)→移轉策略
  • Blake-Mouton (9,1)=職責管理式(高業績低關心人)
  • 觸角/近因效應=考核前發生的事件影響評分
  • 集權式優點=減少工作重複及成本
  • PDCA Check=查核/稽核（非計畫、非執行）
  • 跡近錯失=及時介入使意外未真正發生
  • 通報系統=匿名+自願+保密+不究責+共同學習
  • RCA=回溯式（非預應式）；分析根本原因；組織小組
  • Benner五階段：生手→進階學習者→勝任者(2-3年)→精通者→專家
  • 損益平衡點 = 固定成本 ÷ (1 - 變動成本率)
